# Student Project
هذا المشروع جزء من نشاط تعاوني بين الطلاب لتعلم العمل الجماعي باستخدام GitHub.

## المهام
- تصميم الواجهة.
- كتابة الكود الخلفي.
- توثيق المشروع.

## كيفية المساهمة
1. قم باستنساخ المستودع.
2. أنشئ فرعًا جديدًا.
3. أضف تعديلاتك وقم برفعها.
4. أرسل طلب دمج (Pull Request).
